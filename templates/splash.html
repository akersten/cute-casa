Open Source

We are committed to open source development — the CuteCasa software is and always will be free. CuteCasa is developed under the MIT License. The project can be found, forked, and fiddled with on the CuteCasa GitHub page. Any feature requests or bug reports should be filed as issues against the project on GitHub.
